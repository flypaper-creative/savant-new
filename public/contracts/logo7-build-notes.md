# Savant Logo 9 Runtime Direction

This site is no longer targeting "stylized browser logo."
It is targeting "real hero asset in a browser shell."

## Required outcome
- Loader ring looks like a chrome/gold portal object
- Center logo reads like a real faceted obsidian object
- Emissive seams travel through the mark and the ring
- Entire UI inherits the same material system

## Site-wide style transfer
- Top bar, cards, HUD, and panels should inherit the obsidian/chrome/gold language
- Magenta/cyan legacy accents should be removed or greatly reduced
- The loader should feel like the site’s DNA, not a separate effect

## Runtime rule
If the GLB and texture pack are missing, do not attempt "fake photoreal" by endlessly decorating the SVG.
Use a strict fallback and keep the shell clean.
